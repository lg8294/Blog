iOS中高级开发有什么常用的工作流和工具？

作者：coder yi
链接：http://www.zhihu.com/question/50905679/answer/123285432
来源：知乎
著作权归作者所有，转载请联系作者获得授权。

分为两部分 日常开发的效率工具 和 iOS开发的效率工具

日常开发的效率工具

  • HomeBrew + Cakebrew : macOS 不可或缺的套件管理器 ，Cakebrew则是 Homebrew GUI 应用
  • SourceTree + gitflow : SourceTree是一个很好的git客户端,git-flow 是一个 git 扩展集，可以看看git-flow的工作流，git-flow 备忘清单
  • sublime + packagecontrol : sublime文本编辑器 ，packagecontrol则是sublime的包管理工具
  • shadowsocks + surge : 代理工具，shadowsocks也有自己的客户端，但是搭配surge用起来很更爽
  • Zsh + Oh My Zsh : 用Oh My Zsh配置你的Zsh
  • slack + slackin : slack是一个团队交流工具,加上slackin这个小工具则能够开放进入slack org的权限。
  • macports : 软件包管理系统，跟HomeBrew差不多
  • hopper : Mac的反编译软件
  • Alfred : OS X 效率启动器,Alfred 有很多的workflow
  • dash : API文档浏览器,代码片段管理工具
  • Cornerstone ：SVN的客户端
  • Charles ：抓包代理调试工具、类 Fiddler
  • WireShark : 可以抓取TCP UDP 语音等各种包，Mac工具，比Charles强大
  • Postman / Paw : API 测试工具
  • VisualJSON ：很好的JSON工具
  • Monodraw ：基于文本的流程图工具
  • Sip ：抓颜色
  • Mou/MacDown : Markdown编辑器
  • MindManager ：思维导图
  • MWeb : Markdown编辑器、静态网站博客编辑器

iOS开发的效率工具

  • NWPusher / Knuff : 可以自己发送APNs，方面测试推送服务
  • Cycript : 运行时工具，分析和修改iOS程序的运行时
  • SQLiteManager/MesaSQLite : 管理本地电脑上的SQLite数据库
  • PonyDebugger : 一个远程调试工具包，通过使用Chrome开发者工具来调试iOS应用的网络流量和数据存储
  • Reveal : iOS程序界面调试工具
  • iFunBox ：iOS文件管理工具
  • CocoaPods-app : CocoaPods客户端
  • AirServer ： AirPlay 屏幕镜像工具
  • Resign ： 重签名ipa文件的工具
  • Unused ：检查Xcode没用到的文件
  • WWDC ： 观看WWDC的客户端
